# 🐍 Snake Water Gun Game (Python)

A simple command-line game built in Python where you play Snake, Water, Gun against the computer.

---

## 🎮 How to Play

- 0 = Snake
- 1 = Water
- 2 = Gun

Rules:
- Snake beats Water
- Water beats Gun
- Gun beats Snake

---

## 🚀 Features

- Player vs Computer gameplay
- Score tracking (Wins, Losses, Draws)
- Loop to play multiple rounds
- Input validation

---

## 🧠 Concepts Used

- Python loops
- Dictionaries
- Lists (game matrix)
- Random module
- Conditional logic

---

## ▶️ Run the Game

```bash
python main.py
